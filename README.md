# 🚀 Control-M Log Analyzer with AI

✨ An AI-powered web application to analyze Control-M logs, detect errors, and provide intelligent root cause analysis with solutions.

---

## 🎯 Key Features

- 📂 Upload and analyze Control-M log files
- 🔍 Automatic detection of Errors, Warnings & CTM issues
- 🤖 AI-powered root cause analysis (OpenAI integration)
- 🧠 Structured issue → description → solution format
- 🔗 BMC Knowledge Base search integration
- 🎨 Clean and professional UI dashboard

---

## 🛠️ Tech Stack

- **Frontend:** React.js  
- **Backend:** Node.js, Express.js  
- **AI Integration:** OpenAI API  
- **Scripting:** Python (for advanced log parsing / automation)
